## Arquitetura da autenticação Inter (descoberta importante)

O Inter dá certificados como **PEM separados** (.crt + .key), mas o SDK oficial espera **.pfx**.
Nossa implementação ignora o SDK e usa `requests` diretamente:
```python
requests.post(url, data=..., cert=(cert_path, key_path))
```
Isso é mais simples, mais confiável em Vercel serverless, e sem dependência de GitHub installs.
