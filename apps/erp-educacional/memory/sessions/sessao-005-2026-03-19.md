## Resumo
Construção do módulo de Configuração da IES com abordagem 100% IA Native.

## Diretriz Nova: IA NATIVE
Marcelo definiu que o sistema deve ser **100% IA Native**:
- Assistente IA embutido em todas as páginas
- Auto-preenchimento inteligente (CNPJ → dados da instituição)
- Sugestões e validações em linguagem natural
- Checklist guiado com ajuda contextual

## O que foi feito

### Componentes IA Native
1. **AIAssistant.tsx** — Chat flutuante com:
   - Mensagens contextuais por módulo
   - Sugestões rápidas (chips)
   - Respostas contextuais sobre diploma digital
2. **SmartCNPJInput.tsx** — Input inteligente:
   - Máscara automática de CNPJ
   - Busca via BrasilAPI com um clique
   - Auto-preenche nome, endereço completo
   - Feedback visual (verde = sucesso)

### API Routes
3. **api/cnpj/[cnpj]** — Proxy para BrasilAPI (dados CNPJ)
4. **api/instituicoes** — CRUD (GET lista, POST criação)

### Página Principal
5. **admin/configuracoes** — CRUD completo com:
   - Checklist IA (mostra o que falta configurar)
   - Lista de instituições com badges de tipo
   - Modal de cadastro com 3 seções (Dados, Endereço, Credenciamento)
   - Integração com SmartCNPJInput e AIAssistant
