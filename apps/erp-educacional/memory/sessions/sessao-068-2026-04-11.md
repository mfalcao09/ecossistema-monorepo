## Bugs Corrigidos

### Bug 1: Assinantes "Nenhum assinante configurado"
- **Causa:** FormularioRevisao tem seção Assinantes (adicionada em sessão paralela), mas página de revisão nunca buscava `/api/assinantes` para popular `dadosRevisao.assinantes`
- **Fix:** useEffect na página de revisão busca `/api/assinantes`, filtra ativos, separa eCPFs + eCNPJ, injeta via `setDadosRevisao(prev => ...)`
- **Guard:** pula se `dadosRevisao.assinantes.length > 0` (dados_confirmados já salvos)

### Bug 2: Confiança 0%
- **Causa:** RPC `update_extracao_with_audit` não incluía `confianca_geral` no SET do UPDATE — campo ficava com default 0
- **Fix DB:** RPC agora calcula `AVG(por_arquivo->>'confianca_geral')` antes do UPDATE e grava na coluna
- **Retrofix:** UPDATE retroativo em todas as sessões existentes → ~97%
