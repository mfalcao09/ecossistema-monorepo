- `src/app/api/diplomas/[id]/documentos/assinar/route.ts` — POST dispara assinatura
  - Valida signatários com e-mail (obrigatório para BRy)
  - Baixa PDF via arquivo_path (storage) ou arquivo_url (fallback)
  - Marca `enviado_assinatura` + salva `bry_document_id`
- `src/app/api/webhooks/bry-assinatura-pdf/route.ts` — recebe callback BRy
  - Correlaciona por `externalId` ("{diploma_id}:{tipo}") ou `bry_document_id`
  - CONCLUIDO → status=`assinado`, salva `arquivo_assinado_url`
  - CANCELADO/EXPIRADO → status=`erro`
  - Verifica se todos docs assinados → avança diploma para `aguardando_envio_registradora`
