### Fix Vercel deploy ERROR

**Causa raiz:** `vercel.json` tinha `"functions": {"api/financeiro/*.py": {"runtime": "@vercel/python"}}` — Vercel rejeita runtime sem versão pinada.

**Fix aplicado (commit `3082329`):** Removido o bloco `functions` inteiramente. Os scripts Python (`emit-boletos.py`, `payment-webhook.py`) são serviços Railway standalone — não devem ser serverless functions Vercel.

**Resultado:** Deploy READY com `lambdaRuntimeStats: {"nodejs":9,"python":2}` — Vercel subiu os Python automaticamente com runtime default.
