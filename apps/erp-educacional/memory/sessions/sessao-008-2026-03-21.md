### 6. Correção do Erro de Build — Dependência Circular CSS (PRINCIPAL DESTA SESSÃO)
**Arquivo:** `src/app/globals.css`

**Problema:**
- Deploy `dpl_BU9A8tmWn5n4j9z5QrgWHp9bEdvY` (commit `6de2d57`) e `dpl_FofhvjNWdrKFeCkHmijsLEuoZYhj` (commit `d57ccea`) falharam com:
  ```
  ./src/app/globals.css:67:3
  Syntax error: You cannot `@apply` the `text-gray-400` utility here because it creates a circular dependency.
  ```
- Causa: `@apply text-gray-400` dentro de seletor `.dark .text-gray-400` = loop infinito no PostCSS

**Solução aplicada:**
- Substituídos TODOS os `@apply` dentro de seletores dark mode por valores CSS brutos (RGB)
- Exemplo: `.dark .text-gray-400 { color: rgb(107 114 128) !important; }`
- Os `@apply` no tema claro (`body`, scrollbar) foram mantidos — esses NÃO causam circular dependency
