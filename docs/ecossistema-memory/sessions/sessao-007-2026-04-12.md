## Aprendizado técnico
- Skills Cowork devem ser **pastas** com `SKILL.md` (frontmatter YAML obrigatório)
- Empacotamento via `python -m scripts.package_skill` gera arquivo `.skill` (zip)
- `.skill` é instalado diretamente via `present_files` → botão "Save skill" no Cowork
- Diretório `/sessions/.../mnt/.claude/skills/` é **read-only** — skills criadas em `/tmp/` e output em pasta do usuário
