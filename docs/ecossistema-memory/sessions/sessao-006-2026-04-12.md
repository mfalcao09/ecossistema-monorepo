## Aprendizado técnico
- `figma-developer-mcp` requer flag `--stdio` para rodar em modo MCP (stdio transport)
- Sem `--stdio`, o pacote inicia em modo HTTP server e o Claude Code não consegue conectar
- Notion MCP usa header `OPENAPI_MCP_HEADERS` com Authorization + Notion-Version obrigatório
