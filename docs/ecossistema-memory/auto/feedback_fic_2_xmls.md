A FIC como EMISSORA gera apenas 2 XMLs, não 3:
1. DocumentoHistoricoEscolarFinal — histórico completo com disciplinas
2. DocumentacaoAcademicaRegistro — dados privados + rito de emissão

O 3º XML (DiplomaDigital) é gerado pela REGISTRADORA (UFMS, código 694) após o registro.

**Why:** Marcelo corrigiu em 30/03/2026. O próprio cabeçalho do gerador.ts já documentava isso. Sempre referenciar como "2 XMLs da emissora".

**How to apply:** Em qualquer menção a XMLs do diploma, dizer "2 XMLs" (não 3). O pacote para a registradora contém 2 XMLs assinados + documentos complementares. A registradora gera o DiplomaDigital após registro.
