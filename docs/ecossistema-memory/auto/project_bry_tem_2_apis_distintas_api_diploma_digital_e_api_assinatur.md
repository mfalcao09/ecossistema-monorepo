BRy oferece dois produtos/endpoints separados: 1) API Diploma Digital (apenas XMLs do Diploma Digital MEC) — endpoint: https://api-assinatura.bry.com.br/api-diploma-digital, padrão XAdES AD-RA, ICP-Brasil A3, já integrada no projeto. 2) API Assinatura Digital (todos os demais documentos da IES — histórico PDF, declaração matrícula, atestados, contratos) — endpoint: https://api-assinatura.bry.com.br/api-assinatura-digital, AINDA NÃO INTEGRADA. Toda documentação emitida pela IES deve ser digitalizada e assinada via BRy. NÃO confundir com Lei 14.063 (eletrônica simples/avançada vs digital qualificada).
