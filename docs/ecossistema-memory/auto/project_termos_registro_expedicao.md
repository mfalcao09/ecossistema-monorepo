**Fato:** O fluxo do diploma exige 2 documentos de encerramento para envio à IES Registradora: **Termo de Registro** e **Termo de Expedição**, ambos em PDF, assinados via API BRy Signer.

**Why:** São emitidos como documento de encerramento do processo após a geração e assinatura dos XMLs + conversão PDF/A dos documentos do aluno. Fazem parte do pacote enviado para a registradora.

**How to apply:** Quando planejar a Sprint de encerramento do processo/pacote ZIP para registradora, incluir esses 2 documentos. Template a criar baseado no modelo físico da FIC que Marcelo possui. Registrado em `memory/projects/termos-registro-expedicao.md`.
