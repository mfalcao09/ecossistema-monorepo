Documentos enviados na Fase 1 (upload para extração IA) NÃO devem ser enviados automaticamente para o acervo acadêmico. Nem todos os documentos do upload são relevantes para o acervo. Na Fase 5 (Digitalização e Acervo), exibir lista com checkboxes para o operador selecionar quais docs vão para o acervo. Também permitir adicionar novos documentos nessa etapa. Nunca assumir que todo upload da Fase 1 é relevante para o acervo.
