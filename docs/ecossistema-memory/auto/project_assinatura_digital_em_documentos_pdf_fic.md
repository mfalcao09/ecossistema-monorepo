Documentos PDF FIC terão 3 elementos de assinatura: (1) Barra lateral direita todas páginas — texto vertical rotacionado URL de verificação, estilo Registro de Imóveis; (2) Box assinatura ao final modelo dinâmico A/B — se couber na última página→inline compacto (estilo UNOESTE), se não→página dedicada (estilo OAB). Contém QR Code, signatários, E-CNPJ, código verificação, URL pública; (3) Assinatura digital docs é diferente da assinatura do diploma — APIs diferentes dentro BRy. How to apply: prever seção configuração de assinatura de documentos similar à de diplomas. Box é placeholder até integração BRy.
