A FIC como EMISSORA gera apenas 2 XMLs: 1) DocumentoHistoricoEscolarFinal (histórico completo com disciplinas) e 2) DocumentacaoAcademicaRegistro (dados privados + rito de emissão). O 3º XML (DiplomaDigital) é gerado pela REGISTRADORA (UFMS, código 694) após o registro. Em qualquer menção a XMLs do diploma, dizer "2 XMLs" (não 3). O pacote para a registradora contém 2 XMLs assinados + documentos complementares.
