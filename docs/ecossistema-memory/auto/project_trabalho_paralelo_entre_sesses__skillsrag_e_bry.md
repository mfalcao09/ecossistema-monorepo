## Por que isso importa

Quando Marcelo diz "vou abrir outra sessão para X enquanto trabalhamos em Y", significa que ele está paralelizando o desenvolvimento. Cada sessão cuida de uma frente separada.

**Why:** Maximizar velocity sem bloquear uma frente pela outra (BRy depende de decisões externas, Skills+RAG é autossuficiente).
