Toda a grade curricular (disciplinas, notas, carga horária, docentes, atividades complementares, estágios) deve ser extraída do processo de importação que PRECEDE o formulário. No formulário, esses dados aparecem pré-preenchidos para revisão. Campos que a importação não conseguiu preencher são DESTACADOS VISUALMENTE (amarelo) para preenchimento manual (ex: titulação do docente). No fluxo, a importação de documentos/planilha acontece ANTES do formulário abrir. O formulário recebe os dados já extraídos.
