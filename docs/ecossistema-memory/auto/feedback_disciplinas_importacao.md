---
name: Disciplinas vêm da importação prévia
description: Disciplinas, notas, CH e docentes devem ser extraídos da importação que precede o formulário, não preenchidos manualmente
type: feedback
---

Toda a grade curricular (disciplinas, notas, carga horária, docentes, atividades complementares, estágios) deve ser extraída do processo de importação que PRECEDE o formulário. No formulário, esses dados aparecem pré-preenchidos para revisão. Campos que a importação não conseguiu preencher são DESTACADOS VISUALMENTE (amarelo) para preenchimento manual (ex: titulação do docente).

**Why:** Marcelo definiu em 28/03/2026. O processo de importação é a fonte primária para dados curriculares. Digitar manualmente ~50 disciplinas seria inviável.

**How to apply:** No fluxo, a importação de documentos/planilha acontece ANTES do formulário abrir. O formulário recebe os dados já extraídos. UI destaca em amarelo campos pendentes.
