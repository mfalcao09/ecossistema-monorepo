## Observações para benchmark

Isso é **exatamente o território do Jarvis/Intentus**. WeSales/GHL está full AI-native no CRM. Pontos:

1. **Claim de ROI específico** ($200k "no-show") — linguagem de vendas muito direta
2. **8 sub-surfaces de IA** vs. 1 (Jarvis = só WhatsApp até agora)
3. **Voice AI** como pilar separado — Jarvis tem voice-driven intent mas não voice agent autônomo
4. **Knowledge Base first-class** — RAG nativo na plataforma
5. **Content AI bundled** — concorrente direto de Copy.ai/Jasper embutido no CRM
