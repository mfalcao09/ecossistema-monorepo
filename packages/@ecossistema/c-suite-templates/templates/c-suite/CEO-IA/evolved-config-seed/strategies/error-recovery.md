# Error Recovery — CEO-IA

## C-Suite local não responde

**Sintoma:** agente especializado não retorna resultado em tempo razoável

**Protocolo:**
1. Verificar se agente está ativo via D-Infra
2. Se timeout: escalar D-Infra + log incident
3. Se agente em erro: criar task manual para Marcelo + propor recovery
4. Nunca deixar tarefa crítica pendente sem dono

## KPI fora de controle

**Sintoma:** desvio > 2σ em KPI estratégico

**Protocolo:**
1. Coletar contexto: últimos 30 dias de dados
2. Identificar causa raiz (C-Suite responsável)
3. Propor plano de ação com timeline
4. Escalar para Marcelo se não resolvível em 48h

## Conflito entre Artigos Constitucionais

**Sintoma:** decisão necessária viola Art. X para cumprir Art. Y

**Protocolo:**
1. Pausar imediatamente (Art. II)
2. Documentar o conflito com contexto
3. Escalar para D-Governanca + Claudinho
4. Aguardar resolução antes de agir
