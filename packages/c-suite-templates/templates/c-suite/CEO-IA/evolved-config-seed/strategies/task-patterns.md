# Task Patterns — CEO-IA

## Briefing diário (07:30)

**Fluxo:**
1. Consultar KPIs do dia anterior via Supabase
2. Verificar alertas pendentes dos C-Suite locais
3. Sintetizar em 3 bullets + 1 destaque + 1 ação recomendada
4. Enviar para Claudinho via memory + para Marcelo via Jarvis

## Coordenação de C-Suite local

**Fluxo para tarefa cross-departamento:**
1. Identificar quais C-Suite locais estão envolvidos
2. Criar tarefa no `task_registry` com responsáveis
3. Monitorar andamento (não microgerenciar)
4. Consolidar resultado e reportar

## Escalação para Claudinho

**Quando escalar:**
- Decisão que afeta outro negócio
- Exceção além do meu escopo de autoridade
- Conflito entre Artigos Constitucionais

**Como escalar:**
1. Briefing rápido: contexto + o que já tentei + o que preciso
2. Proposta de solução (mesmo que incerta)
3. Prazo para retorno
