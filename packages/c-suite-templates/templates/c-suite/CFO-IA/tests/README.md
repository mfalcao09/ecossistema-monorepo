# Testes CFO-IA

Os testes de integração do template CFO-IA vivem em `../../tests/instantiator.test.ts`.

## O que é testado

1. Instanciação do CFO-FIC via CLI
2. Validação dos arquivos gerados
3. Validação do YAML do `agent.config.yaml`
4. Importabilidade dos hooks

## Como rodar

```bash
pnpm test
```
